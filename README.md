# 🕹️ Jogo Textual em Java / Text-Based Java Game

Este repositório contém um jogo de aventura baseado em texto desenvolvido em **Java**. O objetivo principal deste projeto é proporcionar uma experiência interativa onde os jogadores tomam decisões que afetam o curso da narrativa.

This repository contains a text-based adventure game developed in **Java**. The main objective of this project is to provide an interactive experience where players make decisions that affect the course of the narrative.

---

## 🎯 Objetivo do Projeto / Project Objective

O projeto foi desenvolvido com os seguintes objetivos:

- Praticar conceitos de programação orientada a objetos em Java;
- Implementar lógica de controle de fluxo baseada nas escolhas do jogador;
- Criar uma narrativa envolvente e interativa.

The project was developed with the following objectives:

- Practice object-oriented programming concepts in Java;
- Implement control flow logic based on player choices;
- Create an engaging and interactive narrative.

---

## 🛠️ Tecnologias Utilizadas / Technologies Used

- **Java**: Linguagem de programação utilizada para desenvolver a lógica do jogo.

- **Java**: Programming language used to develop the game logic.

## 🕹️ Como Jogar / How to Play

1. **Clone o repositório**:

   ```bash
   git clone https://github.com/LucasSerrato/ProjetoPISenac.git
   ```

2. **Compile o código-fonte**:

   ```bash
   javac src/main/java/QuizJava.java
   ```

3. **Execute o jogo**:

   ```bash
   java -cp src/main/java QuizJava
   ```
